Quick Start Guide

1Purchase a USB drive (8GB minimum)

2Download RAMSAFE.iso from ramsafe.org

3Use RUFUS (Windows) or Startup Disk Creator (Linux) to flash the ISO

4Reboot machine and boot from USB (F12 on Dell)

5Select "Try" (no installation needed)

6Begin investigation work - everything runs in RAM

7Reboot and remove USB when done

Note: Does not work on Apple M-series chips (ARM-based)

Hash Analysis Tools

Standard Hashes

  • MD5: 128-bit hash (legacy, not secure)
  • SHA-1: 160-bit hash (deprecated)
  • SHA-256: 256-bit hash (recommended)
  • SHA-512: 512-bit hash (most secure)

Fuzzy Hashes (ssdeep)

  • Compare similar files even if slightly modified
  • Similarity score (0-100)
  • Useful for detecting altered content

Hash Commands

# Standard hashes md5sum image.jpg sha256sum image.jpg sha512sum image.jpg # Fuzzy hash ssdeep image.jpg # Compare two URLs ssdeep_compare_urls.sh url1 url2 # Compare two files ssdeep_compare_files.sh file1.jpg file2.jpg
